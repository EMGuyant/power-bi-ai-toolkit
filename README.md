# Power BI’s AI Toolkit

**Master the Art of AI-Assisted Development — Modernizing the Way You Build.**

This repository is a hands-on supplement to the **Power BI’s AI Toolkit** series on [ethanguyant.com](https://ethanguyant.com). As the business intelligence landscape shifts toward AI-augmented workflows, this blog series and toolkit offer practical patterns, prompt strategies, and quickstart guides to help you build faster and smarter.

Each entry in this repository includes a working `.pbix` file, sample data, and documentation on the AI prompts used to generate or debug the included logic.

## 🛠 How to Use

1. **Clone or download** this repository to your local machine.
2. **Navigate** to the specific scenario folder you wish to explore.
3. **Open** the provided `.pbix` file in Power BI Desktop.
4. **Review** the markdown files within each folder to see the specific AI prompts and "Before/After" logic used in that lesson.

## 🔑 Prerequisites

To get the most out of this toolkit, you should:
* Have a basic understanding of Power BI and DAX (Data Analysis Expressions)
* Install Power BI Desktop installed on your computer. You can download it for free from the [official Microsoft website](https://powerbi.microsoft.com/en-us/desktop/).
* Have access to **Copilot for Power BI Desktop** to test the prompts provided.
* Have the latest version of **Power BI Desktop** installed.
